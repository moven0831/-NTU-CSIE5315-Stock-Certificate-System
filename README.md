# NTU CSIE5315 Stock Certificate System

## Quick Demo
1. **Git clone this repository**

2. **install required modules**
    if yarn isn't install, enter `npm install yarn -g` to get yarn
    ```node
    yarn install
    ```
3. **Start Web App**
    ```node
    yarn start
    ```
4. **Check it on your browser:** http://localhost:3000
